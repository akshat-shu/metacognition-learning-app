### What did our workflow reveal about collaboration and AI reliance?

Teams should have code, Git activity, prompts and some friction. This is when teamwork becomes visible.

Students should notice that teamwork is not just dividing tasks. It is maintaining shared understanding.

They should also notice that AI changes collaboration. One person can generate a lot of code quickly, but that can create new problems if others do not understand it.

### Evidence to inspect

Ask teams to look at:

- commit history
- branch history
- pull requests
- unresolved merge conflicts
- Claude prompts
- code that only one person understands
- features built in parallel
- decisions made without the full team
